# Admin

## Hardware
* LED Strip (WS2801) 
Am einfachsten sind die Treiber WS2801, es gehen aber auch andere. 

## Software
Der Treiber würde dann auf Stufe „Strip“ implementiert. Es ist am einfachsten immer gleich den ganzen Strip neu zu setzen.
